# playground

<img src="icon.svg" width="64" height="64" alt="playground icon">

Standalone HTML canvas games and simulations. No build step, no dependencies, no server needed.

## Apps

| File | Title | Description |
|------|-------|-------------|
| `index.html` | Hello World | The central hub. Navigation portal with dynamic background gradients. |
| `balls.html` | Ball Pit Simulator | Physics sandbox with bouncing balls. Adjust gravity and bounciness. |
| `shoot.html` | Slingshot 3D | 3D physics game. Topple brick buildings with a slingshot and trajectory guide. |
| `marble.html` | Marble Machine 2D | Creative sandbox. Build and edit marble runs with rails, belts, and paddles. |
| `liquid.html` | Liquid Simulation | PBD particle simulation. Compare behaviors of water, oil, and honey. |
| `mechanics.html` | Mechanics Simulation | Linkage CAD tool. Design and animate complex mechanisms like Jansen's walker. |

## Usage

Open any `.html` file directly in a browser. Each app contains a "Home" link to return to the hub. 

### Key Features
- **Zero Dependencies**: Pure HTML5, CSS3, and Vanilla JavaScript.
- **Persistence**: Many apps (like Marble and Mechanics) save your designs to LocalStorage or allow JSON export.
- **Interactive**: Real-time controls for physics parameters in every simulation.
