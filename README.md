# playground

<img src="icon.svg" width="64" height="64" alt="playground icon">

Standalone HTML canvas games. No build step, no dependencies, no server needed.

## Apps

| File | Title |
|------|-------|
| `index.html` | Hello World (hub) |
| `balls.html` | Ball Pit Simulator |
| `shoot.html` | Slingshot 3D |
| `marble.html` | Marble Machine 2D |

## Usage

Open any `.html` file directly in a browser. `index.html` links to all apps.
